üöÄ Getting Started: The Perfect Combo

For most developers, we recommend starting with this powerful combination of 4 servers:

The Essential Developer Stack

# Start all 4 essential servers
docker-compose up -d

docker-compose.yml:

version: '3.8'
services:
  mcp-github:
    image: mcp/github-server
    ports: ["3001:3000"]
    environment:
      - GITHUB_TOKEN=${GITHUB_TOKEN}
      
  mcp-docker:
    image: mcp/docker-server
    ports: ["3002:3000"]
    volumes:
      - /var/run/docker.sock:/var/run/docker.sock
      
  mcp-postgres:
    image: mcp/postgres-server
    ports: ["3003:3000"]
    environment:
      - POSTGRES_HOST=${POSTGRES_HOST}
      - POSTGRES_USER=${POSTGRES_USER}
      - POSTGRES_PASSWORD=${POSTGRES_PASSWORD}
      
  mcp-slack:
    image: mcp/slack-server
    ports: ["3004:3000"]
    environment:
      - SLACK_BOT_TOKEN=${SLACK_BOT_TOKEN}

With this setup, you'll have AI-powered code management, container orchestration, database operations, and team communication - covering 80% of common development workflows!
